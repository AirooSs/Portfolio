# Landing Page

Proyecto desarrollado con Angular 21.

## Requisitos

- Node.js
- Angular CLI

## Instalación

Instalar dependencias:

```bash
npm install
Ejecutar proyecto

Iniciar servidor de desarrollo:

ng serve

Abrir en el navegador:

http://localhost:4200
Build de producción
ng build

Los archivos compilados se generan en la carpeta dist/.

Tecnologías utilizadas
Angular
TypeScript
HTML
CSS / SCSS
BootStrap
